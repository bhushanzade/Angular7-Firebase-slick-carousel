How I Implement slick-carousel


1) Install ngx-slick-carousel using
npm install ngx-slick-carousel --save

2) Install Jquery using
npm install jquery --save

3) Install slick-carousel using
npm install slick-carousel --save

4) Add Dependency in angular.json file
Include this in style
"styles": [
...
"node_modules/slick-carousel/slick/slick.scss",
"node_modules/slick-carousel/slick/slick-theme.scss",
...br
]

Include this in scripts
"scripts": [
...
"node_modules/jquery/dist/jquery.min.js",
"node_modules/slick-carousel/slick/slick.min.js",
...
]
5) Add Code in html component file from
https://www.npmjs.com/package/ngx-slick-carousel
6) Add ngx-slick-carousel module into app module.
7) Follow the steps which is given in below link...
8) Done.

How I Use Firebase


1) Install Firebase using
npm install firebase @angular/fire --save

2) Add Firebase Modules into app module
3) Create Firebase project into firebase console.
4) Take configuaretion of firebase and add into angular environment file e.g.
firebase: {{
apiKey: 'your-key',
authDomain: 'your-project-authdomain', databaseURL: 'your-database-URL',
projectId: 'your-project-id',
storageBucket: 'your-storage-bucket',
messagingSenderId: 'your-messaging-sender-id'
}}
5) Create Add and Get Services using firebase services
6) Done.
